<h1> Restaurant Data Analysis</h1>

<h2>Project Overview</h2>
<p>
This project analyzes restaurant data to understand customer preferences, pricing trends, service availability, and geographic distribution. 
The goal is to extract meaningful insights that can help improve business decisions.
</p>

<h2>Key Analysis Performed</h2>
<ul>
  <li>Top cuisines and cuisine combinations</li>
  <li>City-wise restaurant distribution and ratings</li>
  <li>Price range analysis</li>
  <li>Online delivery vs table booking</li>
  <li>Rating and votes analysis</li>
  <li>Sentiment analysis of reviews</li>
  <li>Geographic visualization using maps</li>
</ul>

<h2> Key Insights</h2>
<ul>
  <li>Most restaurants are in the <b>low to medium price range</b></li>
  <li><b>North Indian and Chinese cuisines</b> dominate</li>
  <li><b>Mid-range restaurants</b> offer more online delivery</li>
  <li><b>High-end restaurants</b> focus on table booking</li>
  <li>Ratings are mostly between <b>3 and 4</b></li>
  <li>Restaurants are concentrated in <b>urban areas</b></li>
</ul>

<h2> Tools & Technologies</h2>
<ul>
  <li>Python (Pandas, NumPy)</li>
  <li>Matplotlib & Seaborn</li>
  <li>Folium (Geographic Analysis)</li>
  <li>Excel / Power BI</li>
</ul>

<h2> Conclusion</h2>
<p>
This analysis shows that affordability, cuisine type, and service offerings strongly influence restaurant performance. 
These insights can help businesses improve strategy and customer experience.
</p>
